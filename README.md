# `DJANGO INEC RESULTS`

 Here is the live link for the task: https://bincom-inec-results.herokuapp.com/

## General Usage

> * Ensure that you set the necessary secret keys

```bash
Clone the repository
Create a virtual environment
Install the requirements.txt
Run migrations
Run the server
```
